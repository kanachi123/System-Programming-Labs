| Команда         | Что делает                                      |
| --------------- | ----------------------------------------------- |
| `crontab -e`    | Редактирует crontab текущего пользователя       |
| `crontab -l`    | Показывает текущие задания cron пользователя    |
| `crontab -r`    | Удаляет crontab текущего пользователя полностью |
| `crontab -i -r` | Удаляет crontab с подтверждением                |
| `crontab file`  | Устанавливает crontab из файла `file`           |
